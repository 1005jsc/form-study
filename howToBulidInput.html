step 0

어떤 데이터를 받아와야하는지 잘 머릿속에 정리한다

step 1
form 테그를 만든다


form action="">
/form>
step 2
필요한 input들을 form테그안에 순서에 맞춰 갯수에 맞춰 넣는다, select, textarea도 넣는다

이번의 경우는 6개
form action=""> input type="text" /> input type="text" /> select name="" id=""> option value=""> /option> option value=""> /option> /select> input type="text" /> input type="text" /> textarea name="" id="" cols="30" rows="10"> /textarea> /form>


step 3
각 input테그에 맞는 타입(type="")과 변수이름(name=""), defaultValue, placeholder를 설정해준다

예 : input type="text" /> -> input type="text" name="student's name" defaultValue={student's name}>
이런식으로


form action=""> input type="text" name="name" defaultValue={''} placeholder='name'/> input type="text" name="company" defaultValue={''} placeholder='company'/> select name="theme" id="theme" defaultValue={'light'} placeholder='light'> option value="dark">dark /option> option value="light">light /option> /select> input type="text" name="duty" defaultValue={''} placeholder='duty'/> input type="text" name="email" defaultValue={''} placeholder='email'/> textarea name="message" defaultValue={''} placeholder='tell us about yourself'> /textarea> /form>


step 4

각각에 classname, ref, onClick을 넣어준다

여기서 classname은 input은 input으로 통일, select는 select로 통일, textarea는 textarea로 통일 하는게 좋다 그래야 나중에 스타일 하기 편하다

form action=""> input ref={nameRef}className={styles.input} type="text" name="name" onClick={handleClick}defaultValue={'name'}/> input ref={companyRef}className={styles.input} type="text" name="company" onClick={handleClick}defaultValue={'name'}/> select ref={themeRef}className={styles.select} name="theme" id="theme" onClick={handleClick}defaultValue={'name'}> option value="dark">dark /option> option value="light">light /option> /select> input ref={dutyRef}className={styles.input} type="text" name="duty" onClick={handleClick}defaultValue={'name'}/> input ref={emailRef}className={styles.input} type="text" name="email" onClick={handleClick}defaultValue={'name'}/> textarea ref={messageRef}className={styles.textarea} name="message" onClick={handleClick}defaultValue={'name'}> /textarea> /form>


step 5 (마지막)

form 에 onSubmit={handleSubmit} 을 달고 input type="submit" />도 하나 만들어준다
